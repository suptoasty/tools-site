Courses

There will be multiple computer science classes that you will need to take during your time as a computer science student. Some computer science classes you will be able to choose. Here are some of the core classes that you will take as a computer science student.

  • Programming I
  • Programming II
  • Software Engineering I
  • Object Oriented Programming
  • Software Engineering II
  • Internet Applications
  • Assembly Language
  • Computer Organization and Architecture
  • Data Structures and Algorithms
  • Programming Languages
  • Operating Systems
  • Software Engineering III - Ethics
  • Software Engineering IV - Tools

Your first semester you will most likely be taking Programming I and Software Engineering I. In programming I you will learn the basics of programming. In Software Engineering I you will learn about the principles and basic knowledge of being a computer scientist.
