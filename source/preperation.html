Preparing to be a Computer Scientist

Computer Recommendations

It is recommended that you get a fast computer so that it is easy for you and the people helping you become a computer scientist to work on your computer. I would recommend that you get an i7 processor and an SSD. This will allow you to easily work on the assignments you will be doing here.

Learn about programming

It also would be useful to learn some things about coding before you come. Of course they will teach you all you need to know about programming but it will be good for you to review some stuff so that you will have an easy start to the semester.

Hello World!

An easy program you could start by learning that most people start with is making a program that simply just prints "Hello world!".
